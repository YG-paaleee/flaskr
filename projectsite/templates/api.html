API ENDPOINTS

Base URL

http://127.0.0.1:5000/api

All endpoints support ?format=json (default) or ?format=xml

Students

  • GET /students - List all students
  • GET /student/:id - Get student by ID
  • POST /students - Create student (requires: student_name, course, year_level, email)
  • PUT /student/:id - Update student (editable: student_name, course, year_level, email | id cannot be changed)
  • DELETE /student/:id - Delete student

Teachers

  • GET /teachers - List all teachers
  • GET /teacher/:id - Get teacher by ID
  • POST /teachers - Create teacher (requires: teacher_name, department, email)
  • PUT /teacher/:id - Update teacher (editable: teacher_name, department, email | id cannot be changed)
  • DELETE /teacher/:id - Delete teacher

Grades

  • GET /grades - List all grades
  • GET /grade/:id - Get grade by ID
  • POST /grades - Create grade record (requires: student_id, student_name, course_code, course_name, grade, semester, school_year)
  • PUT /grade/:id - Update grade (editable: student_name, course_code, course_name, grade, semester, school_year | id and student_id cannot be changed)
  • DELETE /grade/:id - Delete grade
